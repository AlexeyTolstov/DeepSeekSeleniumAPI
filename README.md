# 🆓🐳 DeepSeek API

Бесплатное API для DeepSeek, реализованное через Selenium с оберткой на FastAPI.


### ⚙️ Описание

Проект автоматизирует работу с веб-версией DeepSeek и предоставляет API-интерфейс для отправки и получения ответов.


### 🚀 Производительность

|  Операция |⏳ Время|
|---|---|
|Вход в аккаунт|7.2 - 8.5 с|
|Обработка запросов|4.4 - 15.2 с |
|Создание нового чата|0.3 c|


### 📦 Установка зависимостей
```
pip install requirements.txt
```

### ▶️ Запуск
```
python main.py
```

### 🖥️ Запуск через SSH
Если запускаешь без GUI, может понадобиться виртуальный дисплей:

#### Установка

```
sudo apt install xvfb-run
```

#### Запуск:
```
xvfb-run -a python main.py
```

#### Запуск в фоне
```
nohup xvfb-run -a python main.py > output.log 2>&1 &
```
